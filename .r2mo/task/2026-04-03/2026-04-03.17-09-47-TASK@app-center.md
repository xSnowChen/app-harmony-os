应用中心进行改动：
- 只显示 app-center 以外的应用信息
- 桌面默认不再显示其他应用，移除
- 应用中心就像干净桌面，不需要使用下边的按钮来处理打开什么的，当用户直接点击桌面上应用之后可直接处理
- 将整个应用中心改成企业严谨风格，参考企微
- 不同应用采用不同的 SVG 并且持久化呈现

## Changes

**Date**: 2026-04-03

### Changes Made

1. **UI 重构 - 企业风格** (`entry/src/main/ets/pages/Index.ets`)
   - 全文重写，采用企业严谨风格（参考企微），深蓝主色 #1F3D8C，浅灰蓝背景 #F3F6FC，白色卡片+细边框+内阴影
   - 顶部标题区：应用中心大字 + 副标题描述
   - 指标卡：业务应用数 / 已安装数 / 待安装数，三列横排
   - 业务应用区：2列 Grid 卡片布局，每个卡片含图标+名称+描述+状态标签+操作提示
   - 状态栏：实时显示安装状态
   - 说明栏：简洁使用说明

2. **移除 app-center 自身展示** (`entry/src/main/ets/pages/Index.ets`)
   - `ManagedApp[]` 列表从 5 个缩减为 4 个（仅 app-monitor / app-security / app-hello / app-medication）
   - 完全移除 app-center 的 GridItem

3. **移除所有桌面显隐控制** (`entry/src/main/ets/pages/Index.ets`)
   - 删除 `showOnDesktop` 字段
   - 删除 `serviceName` 字段
   - 删除 `COMMAND_SET_SHORTCUT_VISIBLE` 常量
   - 删除 `setDesktopVisibility()` 方法
   - 删除 `supportsDesktopShortcut()` 方法
   - 删除所有桌面入口相关按钮

4. **移除底部操作按钮** (`entry/src/main/ets/pages/Index.ets`)
   - 删除"进入应用/尝试安装"按钮（点击卡片直接触发）
   - 删除"管理应用"按钮
   - 删除"隐藏桌面入口/在桌面显示"按钮
   - 删除"刷新状态"按钮
   - 删除"系统应用管理"按钮
   - 改为点击卡片直接处理：已安装→直接打开，未安装→按需安装

5. **持久化 SVG 图标** (`entry/src/main/resources/base/media/`)
   - `ic_app_monitor.svg` - 监控中心：显示屏+趋势线图形
   - `ic_app_security.svg` - 安全中心：盾牌+锁图形
   - `ic_app_hello.svg` - Hello 应用：对话框+笑脸图形
   - `ic_app_medication.svg` - 用药提醒：药丸+时钟图形
   - 均采用 #1F3D8C 蓝色系，SVG 格式，可被 `$r('app.media.xxx')` 引用

6. **替换 iconGlyph 为 Image 组件** (`entry/src/main/ets/pages/Index.ets`)
   - 删除 `iconGlyph` 字段
   - 删除 `getAppGlyph()` 方法
   - `ManagedApp` 接口改用 `icon: Resource` 类型
   - 每个应用引用 `$r('app.media.ic_app_xxx')` 加载 SVG

7. **资源文件更新**
   - `string.json`: 更新 module_desc / EntryAbility_desc 为"企业"前缀
   - `color.json`: start_window_background 改为 #F3F6FC，与新背景色一致
